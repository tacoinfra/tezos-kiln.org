IMPORTANT

Nomadic Labs has issued the Tezos 8.2 version with a critical patch to the tickets feature of the Edo protocol. To be able to bake after Edo goes live on Saturday 2/13/2021, Bakers and nodes must run version 8.2. Kiln team is working on the version 0.8.5 containing Tezos 8.2 update and will release it as soon as it is ready.

Send an email to joinslack@tezos-kiln.org to join Tezos Baking Slack.
